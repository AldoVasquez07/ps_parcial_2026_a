# Parcial Pruebas de Software
## Ejecutar el proyecto
- Para ejecurar el proyecto se debe ubicar en: '{dir_clonacion}\ps_parcial_2026_a\parcial\'

- Emplear el comando: _mvn test_ ó _mvn clean test_ (En caso de que haya caché de ejecuciones pasadas).
## Versiones del proyecto
- Java: 21
- Maven: 3.9
